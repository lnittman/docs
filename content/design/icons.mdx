## Usage Guidelines

### File Organization
All icons are organized by style/collection in the `/design/icons/` directory:
- Original source files preserved
- Multiple formats available where applicable
- Consistent naming conventions

### Integration
Icons can be referenced directly in your projects:
```jsx
<img src="/design/icons/kare-inspired-icons/icon-folder.png" alt="Folder icon" />
```

### Design Principles
Each icon set follows specific design principles:
- **Kare-inspired**: Pixel-perfect, high contrast, instantly recognizable
- **Teenage Engineering**: Geometric, minimal, functional
- **Rams Ultra-thin**: Maximum reduction, essential forms only
- **Glassy 3D**: Modern depth, subtle shadows, translucency
