### Alcala
A sophisticated serif with display and text variants.

**Variants**: BlackDisplay, Bold, Italic, Roman

### Stuart
A versatile family with Caption, Text, and Titling optical sizes.

**Optical Sizes**:
- **Caption**: Optimized for small sizes (6-8pt)
- **Text**: Perfect for body copy (9-14pt)  
- **Titling**: Refined for large display (18pt+)

## Web Font Loading

### Basic Implementation
```css
@font-face {
  font-family: 'Louize';
  src: url('/fonts/louize/LouizeDisplayTrialCnd-Regular.otf') format('opentype');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}
```

### Variable Font Loading
```css
@font-face {
  font-family: 'Exposure';
  src: url('/fonts/exposure/ExposureTrialVAR.ttf') format('truetype-variations');
  font-weight: 100 900;
  font-display: swap;
}
```

## Typography Guidelines

### Pairing Recommendations
- **Headers**: Louize Display + body in Stuart Text
- **Technical**: TX02 Mono headers + IosevkaTerm code
- **Editorial**: Alcala + Stuart Text
- **Experimental**: Milling + Exposure

### Performance Considerations
- Use `font-display: swap` for better perceived performance
- Subset fonts when possible
- Consider variable fonts for flexibility with smaller file sizes
