## 🎯 Technology Patterns

### Successful Patterns

1. **Monorepo Structure**
   - Turborepo with pnpm workspaces
   - Shared configs and packages
   - Consistent tooling

2. **Type Safety**
   - 100% TypeScript adoption
   - Strict mode enabled
   - Shared type packages

3. **State Management**
   - Clear hierarchy (RSC → SWR → Jotai)
   - Minimal client state
   - Server-first approach

4. **AI Integration**
   - Mastra framework standard
   - Lazy model initialization
   - Agent/tool/workflow pattern

### Anti-patterns To Avoid

1. ❌ Multiple state management libraries
2. ❌ Mixed package managers
3. ❌ Inconsistent TypeScript configs
4. ❌ Direct model initialization in AI services
